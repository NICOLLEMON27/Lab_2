# PROBLEMA 2- El problema de la mochila.

Construir un algoritmo que dado un conjunto de ítems, cada uno con un nombre y
un peso y un valor, y una mochila de capacidad especificada, encuentre la
combinación de ítems que maximiza el valor almacenado en la mochila sin superar
el peso. Suponga que solo se puede colocar un ítem de cada tipo.
